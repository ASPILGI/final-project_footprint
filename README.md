## 담당 역할

### Template

* header, body, bottom 으로 레이아웃 구분
* 상단 메뉴바와 하단을 고정시켜 메인화면을 나타낸다

### 메인페이지

* swiper를 사용하여 한 페이지에 3개씩 출력

### 회원 DB

* MySQL 회원 DB 생성
* 관리자 권한 부여
